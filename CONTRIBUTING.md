# Contributing to puzzlers

Thanks for your interest in contributing!

## Development Setup

1. Clone the repo: `git clone https://github.com/NathanNorman/puzzlers.git`
2. Ensure you have JDK 11+ installed
3. Compile: `javac -d out src/*.java`
4. Run: `java -cp out MyAbcs`

## Pull Request Process

1. Fork the repo and create your branch from `master`
2. Add tests for any new functionality
3. Ensure all tests pass
4. Update documentation if needed
5. Submit a pull request

## Code Style

- Follow existing code conventions
- Keep changes focused and atomic

## What We Accept

- Bug fixes
- New puzzler problems and solutions
- Documentation improvements
- New features (please open an issue first to discuss)
